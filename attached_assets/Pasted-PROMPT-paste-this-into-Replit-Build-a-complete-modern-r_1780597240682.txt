Build a complete, modern real estate website for Mariela Martínez Negocios Inmobiliarios, a real estate agency based in Paraná, Entre Ríos, Argentina. This is a custom platform — no WordPress, no Wix, no third-party CMS.

TECH STACK
Frontend: React + Vite + Tailwind CSS + React Router
Backend: Node.js + Express + PostgreSQL (or SQLite for development)
ORM: Prisma
File uploads: Multer (property photos)
Environment: .env for config

DESIGN SYSTEM
Typography: Montserrat (Google Fonts) — use font-weights 300, 400, 600, 700, 800
Color palette:

White: #FFFFFF
Black: #0A0A0A
Light blue (celeste): #5BB8E8
Deep blue (azul): #1A3A6B

Design direction: Clean, professional, minimal-luxury. Think editorial real estate — generous white space, strong typographic hierarchy, bold section headings in deep blue, celeste used as a sharp accent for CTAs and highlights. No gradients, no shadows on everything — selective, intentional use. Mobile-first responsive layout.
All page text MUST be written in Spanish (Argentina), with precise and professional UX writing. Every label, button, placeholder, heading, description, and microcopy must be in Spanish. Avoid generic filler text — write real, clear, conversion-oriented copy.
